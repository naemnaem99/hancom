# [Tool 01] 손그림 스케치를 UI로 변환하는 AI 툴

> 작성: 2026-07-03

## Q&A

**Q. 페이지 디자인을 손으로 그린 스케치에서 실제 UI로 바꿔주는 AI가 있나?**

있다. 목적에 따라 선택지가 다르다.

**Q. 가장 "실시간"에 가까운 느낌의 툴은?**

**tldraw + makereal** — tldraw 캔버스에 손으로 그리면 버튼 하나로 실제 작동하는 HTML/CSS로 변환된다. 오픈소스라 무료이고, 빠른 프로토타이핑에 적합하다.

**Q. 피그마 AI는 손그림 인식이 되나?**

아직 약하다. 피그마 AI(First Draft 등)는 텍스트 프롬프트 기반이지 스케치 업로드 방식이 아니다.

**Q. 별도 툴 없이 가장 빠르게 시도하려면?**

손그림을 사진 찍어 Claude나 GPT-4o에 올리고 "이걸 HTML/CSS로 구현해줘"라고 하면 바로 코드가 나온다. 지금 이 대화창에서도 가능하다.

---

## 개념 요약

| 툴 | 방식 | 특징 |
|---|---|---|
| **Uizard** | 스케치 사진 업로드 → UI 변환 | 웹/앱 UI 특화, 가장 유명 |
| **tldraw + makereal** | 캔버스에 그리기 → HTML/CSS | 오픈소스, 실시간에 가장 가까움 |
| **Figma AI** | 텍스트 프롬프트 → UI | 손그림 인식 약함 |
| **Visily** | 스케치 사진 → UI | 피그마 export 가능 |
| **Claude / GPT-4o** | 사진 첨부 → 코드 생성 | 별도 툴 불필요, 가장 빠름 |

**추천 흐름**:
```
손그림 사진 찍기 → Claude/GPT-4o에 올려 HTML/CSS 받기 → 피그마로 다듬기
```

**핵심**: 피그마 AI보다 Claude에 사진을 올리는 것이 현재 가장 빠르고 실용적이다.
