### JS 기본 문법

**변수 선언**
```js
let score = 0;       // 나중에 바꿀 수 있는 상자
const btn = ...;     // 한 번 담으면 고정되는 상자
```
- `let` — "새 변수(상자) 만들게" 하는 키워드
- `const` — 한 번 담으면 고정. 요소 참조에 주로 사용
- `=` — "이 값을 상자에 넣어" 라는 뜻. 같다는 비교가 아님

**템플릿 리터럴**
```js
`안녕, ${myName}!`   // ✅ 백틱 + ${}
"안녕," + myName     // 옛날 방식
```
백틱(`) 안에서 `${}` 로 변수를 문자열에 끼워 넣는다.

### DOM 조작

```js
const title = document.querySelector("#title")
```
- HTML에서 `id`로 붙인 이름표를 JS에서 다시 이름표 붙이는 것
- HTML `id`는 HTML/CSS용, `const title`은 JS 안에서 쓰는 이름표
- `#` = id, `.` = class, 태그명 그대로 = 태그 선택

```js
title.textContent = "hello"   // 텍스트만 바꿈
title.innerHTML = "<b>hello"  // HTML 태그까지 삽입 가능
input.value                   // 입력칸에 친 글자를 꺼내는 속성
```

### 이벤트

```js
btn.addEventListener("click", () => {
    // 클릭 시 실행
})
```
- 1번째 인자: 어떤 이벤트인지 (`"click"`)
- 2번째 인자: 그때 실행할 함수 (`() => {}`)
- `()` 안은 보통 비워둠. 클릭 정보가 필요할 때만 `(e)` 넣음

---

### 조건문

```js
if (op.value === "+") { result = x + y; }
else if (op.value === "-") { result = x - y; }
else { result = x / y; }
```
- `===` — 값과 타입이 같은지 비교 (`=` 대입과 다름)
- `select`의 선택값은 `option`의 `value` 속성으로 전달됨

---

### 함수

```js
const multi = (n1, n2) => n1 * n2;
```
- 화살표 함수 `() => {}` 로 선언
- 한 줄이면 `{}` 와 `return` 생략 가능
- `const`로 선언해서 재사용 가능

### JS로 HTML 요소 생성 (08)

```js
const el = document.createElement("span")  // span 태그 새로 생성
el.textContent = "텍스트"                   // 내용 넣기
el.style.left = "30vw"                     // 스타일 적용
document.body.appendChild(el)              // body에 추가
el.remove()                                // 요소 삭제
```
