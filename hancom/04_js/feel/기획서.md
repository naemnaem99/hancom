# feel — 역기획서

> 이미 만들어진 웹페이지를 보고 거꾸로 정리한 기획서.
> 헷갈릴 때 다시 돌아와서 보는 나침반 역할.

## 프로젝트 이름

**feel** (기분 쏘기)

## 한 줄 소개

말하지 못한 감정을 입력하고, 눈처럼 흩날려 떠나보내고, 바구니에 골라 담았다가 불에 태워 비워내는 감정 배출 웹앱

## 목표 (왜 만드나)

- 좋은 감정이든 나쁜 감정이든, 마음속에 있는 걸 일단 "말로 꺼내는" 행위 자체에 의미를 둔다.
- 참지 말고 욕이든 뭐든 꺼내도 괜찮다는 허용적인 톤으로 사용자를 안심시킨다.
- **배출 → 선택 → 소각** 3단계 인터랙션으로, 감정을 느끼고 → 마주하고 → 흘려보내는 과정을 은유적으로 체험하게 한다.

## 핵심 기능 3개

1. **감정 입력 (텍스트 배출)**
   input에 감정을 적고 버튼(ENTER → 외치기 → 토해내기)을 누르면 화면에 눈처럼 흩날리며 떨어진다.
   클릭할 때마다 안내 문구·버튼 문구가 단계적으로 바뀌며, "오늘 기분이 어때요?" → "좋은 일이 있었나요?" → "나쁜 말도 시원하게 해봅시다" 순으로 점점 더 편하게 털어놓도록 유도한다.

2. **바구니로 선택적 담기**
   3단계를 마치면 화면 중앙에 위쪽이 뚫린 픽셀아트 바구니가 등장한다. 드래그해서 떨어지는 글자 중 원하는 것만 입구(상단 중앙 1/3)에 맞춰 담을 수 있다. 가만히 두면 아무것도 안 담기고, 직접 움직여야 캐치된다 — "무의식적으로 다 받아들이는 게 아니라, 내가 고르는" 감각을 준다.

3. **불태워 비우기**
   바구니를 처음 잡는 순간 화면 하단에 불꽃이 등장한다. 담은 감정이 든 바구니를 불 위로 끌고 가면 타서 사라지고, 새 바구니가 곧바로 다시 생성되어 담기 → 태우기를 계속 반복할 수 있다.

## 화면 구성 (어디에 무엇)

| 위치 | 내용 |
|---|---|
| 최상단 | (⬇️ 픽셀 아이콘) 현재 단계 안내 문구 (⬇️ 픽셀 아이콘) |
| 안내 문구 아래 | 감정 입력창(input) + 액션 버튼 |
| 화면 전역 | 버튼 클릭마다 입력한 텍스트가 눈처럼 낙하하는 애니메이션 |
| 화면 중앙 (3단계 이후) | 드래그 가능한 픽셀아트 바구니 + 바구니 하단 안내 문구 |
| 화면 하단 (바구니를 만진 이후) | 불꽃 이미지 + (⬆️ 픽셀 아이콘) "태워버리기" (⬆️ 픽셀 아이콘) |
